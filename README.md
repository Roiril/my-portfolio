# Roilのポートフォリオ

明治大学総合数理学部FMSでHCIを研究し、Web・3D・XRを中心に体験設計と実装を行っています。

Unity、Next.js、Blenderなどを用いて、プロトタイプから映像制作まで幅広く取り組んでいます。
